## Kolorowy pasek postępu
### Rozmiar paska jest w pełni uzależniony od niestandardowej właściwości:
```css
--bar-width
```
### Kolor komponentu można kontrolować za pośrednictwem nazw klas:
* red
* orange
* green
* blue
* violet
* purple
* pink
